# TaskFlow – Gestor de Tareas

## Descripción del proyecto

Este proyecto consiste en una pequeña aplicación web llamada **TaskFlow**, cuyo objetivo es permitir a los usuarios organizar sus tareas de forma sencilla.

La aplicación permite **agregar tareas, marcarlas como completadas y eliminarlas**, todo desde una interfaz simple y moderna.

Este proyecto fue desarrollado como práctica para aprender y comprender mejor el funcionamiento de **HTML, CSS y JavaScript**, especialmente cómo interactúan entre sí para crear aplicaciones web dinámicas.

---

## Objetivo del proyecto

El objetivo principal fue aprender a:

* Crear una estructura web utilizando **HTML**
* Aplicar estilos modernos usando **CSS**
* Utilizar **JavaScript** para agregar interactividad
* Manipular elementos del **DOM**
* Escuchar eventos del usuario como clics y entradas de texto

---

## Qué hace la aplicación

La aplicación permite:

* Escribir una tarea en un campo de texto
* Agregar la tarea a una lista
* Marcar la tarea como completada
* Eliminar una tarea de la lista

Esto ayuda a practicar cómo **JavaScript puede modificar el contenido de una página web en tiempo real**.

---

## Tecnologías utilizadas

Para desarrollar este proyecto utilicé:

* **HTML5** → para la estructura de la página
* **CSS3** → para el diseño, colores y estilo moderno
* **JavaScript** → para la lógica y la interacción del usuario

No se utilizaron frameworks externos, todo fue desarrollado con **JavaScript puro (Vanilla JavaScript)**.

Durante el desarrollo de este proyecto pude entender mejor varios conceptos importantes:

Aprendí a organizar los elementos usando HTML como:

También aprendí a usar **eventos**, como:
